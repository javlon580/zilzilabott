# Zilzila Xavfsizlik Boti

Telegram bot for earthquake safety information based on seismic zones.

## Features

- Users can select their seismic zone (1-9) to get safety advice
- Users can share their location to automatically detect their zone
- Detailed safety recommendations for each zone
- Precautions checklists
- Links to Telegram channels for updates
